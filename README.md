# 🤖 LUNA_V2

### AI Semi-Humanoid Robot (Generation II)


> **LUNA_V2** is the second-generation AI semi-humanoid robot developed by **TechBots-SIT**, featuring a redesigned fully 3D-printed body, enhanced arm articulation, synchronized head-body movement, and improved embedded motion control while retaining AI-driven interaction through Raspberry Pi.

---

## 📖 Overview

LUNA_V2 builds upon the original LUNA platform with significant mechanical, electronic, and software improvements. The robot integrates Raspberry Pi-based AI capabilities with Arduino Mega-based real-time motion control to create a more expressive and capable semi-humanoid robotic platform.

LUNA_V2 introduces dual-arm articulation, synchronized head rotation during turning, smoother servo motion, and an improved mechanical design for enhanced interaction and realism.
